About:
Implementation of a data structure maintaining a total order under order querys
and updates within O(1) worst case complexity presented by T. Kopelowitz in 
"On-line Indexing for General Alphabets via Predecessor Queries on Subsets of an Ordered List".
For the purpose of a bachelor's degree.

Usage:
main uses the structure on a random data.
symulator uses the structure on a provided data.
